======================
Backend
======================

1. masuk ke folder #Backend lewat cmd
2. build dengan command: (pastikan package golang sudang terinstal) [[Bisa instal dari website resmi nya]]
   2.1 (kalau mau deploy ke vps pake command ini) env GOOS=linux GOARCH=amd64 go build -o backend main.go
   2.2 (kalau mau dijalanin di lokal windows pake command ini) env GOOS=windows GOARCH=amd64 go build -o backend.exe main.go
3. kalau ingin menjalankan di lokal tinggal klik dua kali file backend.exe (nanti muncul cmd)
4 kalau ingin mengirimkan ke vps bisa:
   4.1 Login dulu ke vps pake command:  ssh root@103.253.212.21
   4.2 kalau ada pilihan "Are you sure you want to continue connecting (yes/no)?" ketik yes lalu enter
   4.3 masukkan password vps: QOj!HXh9p%4f7N
   (bonus) kalau command error pas jalanin ssh root@103.253.212.21 berarti harus pakai aplikasi tambahan yaitu putty (bisa cari di internet aja instal trus konek ke vps pake ssh)

   4.4 Kalau sudah login bisa hapus langsung executable backend yang udah ada di vps pake:  rm backend
   4.5 lanjut buka cmd baru trus masuk ke folder #Backend (yang nyambung ke vps gausah ditutup)
   4.6 copy / kirim file executable backend yang udah dibuat tadi pake command: scp backend root@103.253.212.21:/root/
   4.7 balik ke cmd yang udah nyambung ke vps trus ketik:  systemctl restart absensi.service
   4.8 klo gak muncul error berati udah aman dan jalan di vps




======================
Frontend
======================

1. Masuk ke folder #frontend lewat cmd
2. ketik:  npm install
3. setelah install selesai bisa login ke vercel pake command:  vercel login
4. nanti muncul ke browser nanti login pake akun github ku aja (nanti chat hehe)
5. klo udah login nanti otomatis di cmd udah autentikasi.
6. Langsung kirim ke vercel pake command:  vercel --prod
